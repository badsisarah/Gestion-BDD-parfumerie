# 🗄️ Application Web de Gestion de Base de Données (Parfumerie)

## 📌 Description
Ce projet consiste à développer une application web permettant de gérer une base de données relationnelle autour d’un système de parfumerie.

---

## ⚙️ Fonctionnalités
- Consultation des données
- Affichage des détails
- Insertion de données
- Modification des données
- Suppression des données

---

## 🗂️ Structure du projet
- Flacon → gestion des flacons
- Magasin → gestion des magasins
- Parfum → gestion des parfums
- Stock → gestion des stocks

---

## 🛠️ Technologies utilisées
- PHP
- SQL (PostgreSQL / MySQL)
- HTML / CSS

---

## 🧠 Compétences développées
- CRUD (Create, Read, Update, Delete)
- Gestion base de données
- Développement web
- Utilisation des sessions PHP
